assertEqual (all data types)

Syntax:

assertEqual( 'erwarteterWert' : 'tatsächlicherWert' [: 'nachricht');

Beschreibung:

Vergleicht die beiden Zeichenketten 'erwarteterWert' und 'tatsächlicherWert' miteinander. Der Testfall gilt als fehlgeschlagen, wenn die beiden Zeichenketten unterschiedlich sind.

Diese Zusicherung akzeptiert alle Datentypen.

Funktionswert:

void

Beispiel 1 (string): 

assertEqual( 'Hello' : 'Hello' );          

// No exception message => OK.        

assertEqual( 'HelloWorld' : 'Hello' );

// Never reached!

Beispiel 2 (timestamp): 

assertEqual( z'2024-09-16-10.25.00.000' : z'2024-09-16-10.25.00.000' );          

// No exception message => OK.        

assertEqual( z'2024-09-16-10.25.00.000' : z'2024-12-31-00.00.00.000' );          

// Never reached!
