Umsteigen von v4.0.4 nach v4.1.0

In dieser Version wurde die Unterstützung für folgende, ältere RPGUNIT Bibliotheken entfernt:

  • Version 1 vom 22.04.2013
  • Version 2 vom 10.10.2016

Darüber hinaus hat sich das Format des User Spaces für die Übermittlung der Testergebnisse an das iRPGUnit Eclipse plug-in geändert und liegt. Die aktuelle Version ist 4. User spaces der Version 3 werden weiterhin unterstützt.

Hinweis: Diese Version erfordert iSphere v3.7.3 oder höher, insofern iSphere eingebunden werden soll.
