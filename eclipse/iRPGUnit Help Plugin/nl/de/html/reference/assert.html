assert (assert)

Syntax:

assert( boolscherAusdruck : 'nachricht' );

Beschreibung:

Prüft den übergebenen boolschen Ausdruck auf true. Der Testfall gilt als fehlgeschlagen, wenn die Auswertung des Ausdrucks false ergibt. In diesem Fall wird nachricht als zusätzliche Information im Testprotokoll ausgegeben.

Beispiel 1: 

assert( 1=2 : 'Obviously wrong' );          

// Never reached!

Beispiel 2: 

assert( rc=0 : 'Unexpected return code. Expected: 0' );          

// Never reached!

Cobol Syntax:

Call "assert" Using
   By Content bResult
   [By Content "message".
