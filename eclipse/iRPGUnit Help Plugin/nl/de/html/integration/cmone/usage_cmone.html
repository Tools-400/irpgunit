Anwendung mit CMOne

Wenn sie CMOne, die Software Change Management Lösung der Task Force IT-Consulting GmbH, einsetzen, können iRPGUnit in CMOne integrieren. Dazu müssen sie lediglich das iRPGUnit CMOne Integration Feature installieren:

Nun können sie eine RPGUnit Test Suite direkt von einem CMOne Arbeitsblatt aus starten:

Hinweis: Die Option Ausführen Unit Test wird nicht angezeigt, wenn das ausgewählte Serviceprogramm nicht den, in den Einstellungen hinterlegten, Anforderungen entspricht.

Beispiel eines CMOne Typcodes für RPGUnit Test Suites:

Typcode: RUNIT
Beschreibung: Unit Test
Wertigkeit: 401
Quellentypcode: *YES
Objekttypcode: *YES
Quellendatei: QRUNIT
Quellenattribut: RPGLE
Objektart: *SRVPGM
Objektattribut: *BLANK
 
Befehl: RUCRTRPG
Parameter TSTPGM: &OL/&EO
Parameter SRCFILE: &SL/&SF
Parameter SRCMBR: &ES
Parameter COPTION: *EVENTF

Der Typcode für Test Suites mit eingebettetem SQL sieht prinzipiell gleich aus. Allerdings sollte sich die Beschreibung unterscheiden und das Quellenattribut muss auf SQLRPGLE gesetzt werden.

Tipp: Deaktivieren sie die Voreinstellung Anzeigen Ergebnissicht, um nach einem Testlauf auf dem CMOne Arbeitsblatt zu verbleiben. Die RPGUnit Ergebnissicht visualisiert das Ergebnis eines Testlauf mit einem grünen oder roten Icon auf dem Tab der Sicht.
